# 🍽️ Reclamações da Merenda Escolar

Este é um site desenvolvido para que alunos possam registrar reclamações e opiniões sobre a comida da escola de forma simples e organizada.

## 📌 Objetivo

O projeto tem como objetivo melhorar a comunicação entre alunos e escola sobre a qualidade da merenda escolar.

Os usuários podem:

- Enviar reclamações
- Escolher categorias de problemas
- Visualizar reclamações enviadas
- Usar o site em computador e celular

---

- HTML5

- JavaScript

## ▶️ Como Executar

1. Baixe os arquivos
2. Abra a pasta do projeto
3. Execute o arquivo `index.html` no navegador

---

## 🌐 Publicação no GitHub Pages

Para publicar:

1. Crie um repositório no GitHub
2. Envie os arquivos do projeto
3. Vá em:
   - `Settings`
   - `Pages`
4. Selecione:
   - Branch: `main`
5. Clique em `Save`

Seu site ficará online.
